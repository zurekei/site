# data/gdp_forecast.csv について

内閣府「経済見通しと経済財政運営の基本的態度」の当初見通し(GDP成長率)と、国民経済計算(SNA)の実績値を年度単位で並べたデータ。

## 収集範囲

- 見通し(forecast_real): FY1980〜FY2025
  - FY1998〜FY2025は内閣府アーカイブの閣議決定版(閣議了解版は同一見通しの前段階のため除外)
  - FY1980〜FY1997は国会会議録の経済演説(下記「出典」参照)
- 見通し(forecast_nominal): FY1980・FY1981とFY1998〜FY2025。FY1982〜FY1997は未収集(下記「未着手」参照)
- 実績(actual_real / actual_nominal): FY1981〜FY2024(FY1980は接続元系列の起点で成長率算出不能)
- FY2025の実績値は年度未終了のため空欄

## 出典

- 見通し(FY1998〜): https://www5.cao.go.jp/keizai1/mitoshi/mitoshikako.html 掲載の各年度PDF/HTML(行ごとのforecast_source_urlを参照)
  - FY2000は現在の内閣府URLが本文消失のスタブページ化していたため、Wayback Machineのアーカイブ経由で確認
  - FY1999の名目値は別添「主要経済指標」表が画像(gif)のため、画像を直接読み取って確認
- 見通し(FY1980〜FY1997): 国立国会図書館「国会会議録検索システム」に収録された**経済企画庁長官の経済演説**(通常国会冒頭の政府four演説の一つ)。政府が閣議決定した当該年度の経済見通しの成長率を、大臣が本会議で読み上げた記録
  - 内閣府のアーカイブページはFY1998始まりで、それ以前の閣議決定文書本体はウェブ上に存在しない。経済演説は全文が恒久URLで公開されており、同一内容が衆議院・参議院の2つの会議録に独立して記録されているため、転記の相互検証ができる
  - 収録した数値はすべて衆議院本会議の会議録から採り、参議院の会議録と全18年分が一致することを確認済み
  - 一部の年度は翌年度の演説が前年度の「当初見通し」を再掲しており、追加の裏取りに使った(例: FY1980の4.8%はFY1981演説の「当初の見通しどおり、四・八％程度」、FY1983の3.4%はFY1984演説の「当初見通しの三・四％程度」、FY1984の4.1%はFY1985演説の「昨年初めの当初見通しの四・一％程度」)
  - FY1997は財務省財務総合政策研究所『平成財政史』が「名目3.1％程度、実質1.9％程度」と記載しており、実質1.9%を外部から確認できた — https://www.mof.go.jp/pri/publication/policy_history/series/h1-12/1_2_2.pdf
- 実績:
  - FY1981〜FY1994: 2015年基準・簡易遡及の参考系列(内閣府ESRI、令和3年12月8日公表) — https://www.esri.cao.go.jp/jp/sna/data/data_list/h27_retroactive/
  - FY1995〜FY2024: 2020年基準・2008SNA・2024年度国民経済計算年次推計(正式系列) — https://www.esri.cao.go.jp/jp/sna/data/data_list/kakuhou/files/2024/2024_kaku_top.html

## 既知の注意点

- **基準改定によるヴィンテージ差**: GDP統計は基準年改定のたびに過去に遡って数値が改定される。本CSVの実績値は2024年度の最新確報(2020年基準)を使用しているため、より過去の時点で公表された数値とは一致しない年度がありうる。実績値側の基準改定履歴までは今回のCSVでは表現していない(将来的に取得日・確報/速報の別を記録する運用に拡張余地あり)。
- **FY1993以前の見通しはGNPベース**: 政府経済見通しの主要指標は、FY1994(平成6年度)の経済演説で初めて「国内総生産の実質成長率」と明記され、FY1993以前の演説は「実質経済成長率」としか述べていない。当時の主要指標は国民総生産(GNP)であり(FY1993については国会質疑でも「ＧＮＰ四百九十五兆三千億円、名目成長率四・九％、実質三・三％」と言及されている)、実績値のGDPとは概念が異なる。FY1980〜FY1993の見通しと実績を比べる際はこの差が乗ることに注意。
- **FY1980〜FY1997のforecast_published_dateは演説日**: 閣議決定日ではない。閣議決定は演説の数日前だが、日付を一次資料で確認できないため演説日を記録している。
- FY2011の見通し文書のみ章立てが「Ⅰ〜Ⅳ」形式(通常は「１〜３」)で、対象年度見通しは「Ⅳ．平成23年度の経済見通し」に該当。
- FY2013・FY2015は閣議決定日が2月(通常は1月)。政権交代直後の年度で決定が遅れたため。

## 未着手(今後の作業)

- **FY1982〜FY1997の名目成長率見通し**(最優先)。実質はFY1980まで遡って収集済みだが、名目は経済演説がFY1982以降ふれておらず、16年分が欠けている。有力な探索先は次の2つ
  - 大蔵省/財務省編『国の予算』各年度版に「政府経済見通し」の主要指標表が載る(財務省『平成財政史』が出所として明示している)。ウェブ公開はなく、国立国会図書館等で現物にあたる必要がある
  - 財務省財務総合政策研究所『平成財政史』(ウェブ公開)は年度ごとの記述の中で名目・実質を併記している箇所があり、FY1997は「名目3.1％程度」を確認済み。他年度も同シリーズを追えば拾える可能性がある
- FY1989とFY1993〜FY1997の消費者物価見通し。経済演説にその年度の消費者物価上昇率の数値がなく、`cpi_forecast.csv` でこの6年分が空欄のまま
- 実績値のヴィンテージ(基準改定)の表現。現状は2020年基準の最新確報で全期間を統一しているため、「当時公表されていた実績」とは異なる。上記「既知の注意点」参照
- 名目/実質の乖離だけでなく、関連指標の追加余地(消費者物価は`cpi_forecast.csv`として追加済み)

## 対応済み(旧「未着手」からの繰り上げ)

FY1980〜FY1997の実質成長率見通し(18年分)は国会会議録の経済演説から収集し、`gdp_forecast.csv` に反映済み。同じ演説から消費者物価の見通しも12年分取れたため `cpi_forecast.csv` にも反映した。

税収見積もり(Phase 2)・出生率(Phase 3)はいずれも実装・公開済み。現在サイトが扱う指標は次の8系列 + 出生率ページ。

- `gdp_forecast.csv`(実質・名目) / `tax_revenue_forecast.csv` / `cpi_forecast.csv` / `unemployment_forecast.csv` / `current_account_forecast.csv` / `bond_issuance_forecast.csv` / `jgb_total_issuance_forecast.csv`
- `fertility_forecast.csv` + `fertility_actual.csv`(専用ページ `/fertility`)
